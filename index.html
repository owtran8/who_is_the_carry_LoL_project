I AM THE MAIN CARRY! Measuring League of Legends Role's Impact in Professional ESPORTS in 2025

Project Overview

This project uses professional League of Legends esports data from Oracle's Elixir to study which roles most often produce the strongest player performance on winning teams.

Instead of defining a carry only by damage or gold, I use a role-adjusted impact score. This means each player is compared to other players in the same role, so supports, junglers, and damage roles can be evaluated more fairly.

Main Question

Among winning teams, which role is most likely to be labeled as the main impact player?

Hypothesis Test

My null hypothesis is that top, jungle, mid, bot, and support are equally likely to be the main impact player among winning teams. This would mean each role should appear as the main impact player about 20% of the time.

My alternative hypothesis is that some roles are more likely than others to be labeled as the main impact player.

I used Total Variation Distance, or TVD, to compare the observed role distribution to the expected equal 20% distribution.

Prediction Problem

For the prediction part of the project, I am building a binary classification model. The model predicts whether a player was their team's main impact player.

The target column is main_carry, where 1 means the player was labeled as the main impact player and 0 means they were not.

Planned Models

My baseline model will use Logistic Regression with simple post-game features. To improve the model, I plan to engineer features such as KDA, kill participation, damage per minute, vision per minute, and role-adjusted performance percentiles.

For the final model, I plan to compare Logistic Regression, Random Forest, and Gradient Boosting using GridSearchCV.
